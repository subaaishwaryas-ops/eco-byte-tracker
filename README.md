# Eco-Byte Tracker 🌱

A Java-based application that calculates digital carbon footprint based on user activity.

## 🚀 Features
- Calculates CO2 emissions from:
  - Emails sent 📧
  - Video streaming 📺
- Displays total carbon footprint
- Shows real-world equivalent (car travel 🚗)
- Provides eco-friendly suggestions

## 🛠️ Tech Stack
- Java
- Object-Oriented Programming (OOP)

## 📊 How It Works
1. User inputs:
   - Number of emails sent
   - Data used for streaming (in GB)
2. System calculates total CO2 emissions
3. Displays impact and suggestions
